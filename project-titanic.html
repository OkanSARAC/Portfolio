Titanic data analysis & prediction

This first project uses the famous Titanic dataset, it involves some data analysis, data cleaning, webscraping and classification for the famous Kaggle competition.
I got a score of 0.79425 (ranked 850th, top 1,69%) with this model on Kaggle. I could try more models and change some of the data but it is pretty good. The goal of this exercise was to train myself and create a good model in my first try and without using insights from previous works, in a real scenario you don't get to resubmit your work multiple times so being able to get a good classification model after a good analysis is key.

→ Analysis and prediction

I also created a dashboard to explain the relation between variables of the dataset so people can understand who was more likely to survive during the disaster. This is my first attempt at a dashboard, I am new to Tableau so I did not use most of the tools that make Tableau so great but I will work on other dashboards for sure.
